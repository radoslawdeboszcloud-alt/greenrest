# Greenrest

Statyczna strona wizytowka dla projektu agroturystycznego `Greenrest`, przygotowana pod GitHub Pages.

## Co jest gotowe

- dwujezyczny one-page: `PL / DE`
- sekcje: hero, opis miejsca, pobyt, galeria, kontakt
- dwa zdjecia JPG z folderu projektu wykorzystane na stronie
- workflow `.github/workflows/deploy-pages.yml` do publikacji przez GitHub Actions

## Czego jeszcze brakuje

- prawdziwych danych kontaktowych
- pelnych danych do Impressum
- ewentualnej konwersji plikow `HEIC` do `JPG` lub `WEBP`

## Publikacja

Po wypchnieciu repo na GitHub:

1. ustaw repo jako `Public`, jesli ma byc dostepne publicznie
2. w `Settings -> Pages` wybierz `GitHub Actions`
3. push na galez `main` uruchomi publikacje
